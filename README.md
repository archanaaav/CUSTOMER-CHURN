# Customer Churn Analysis

## Objective
Performed Exploratory Data Analysis (EDA) on a bank customer churn dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
- Data loading
- Data cleaning
- Missing value check
- Outlier detection using IQR
- Boxplot visualization
- Label Encoding
- One-Hot Encoding

## Outcome
Prepared the dataset for machine learning and gained insights into customer churn patterns.
